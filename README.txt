Hola buenos, dias. Mi proyecto se basa en un e-commerce de venta de ropa para hombre y mujer. Cuenta con productos como remeras, pantalones y zapatos. Una tienda de ropa básicamente.
El proyecto esta trabajado con hooks de react que vimos a lo largo de la cursada, como useState para setear ciertos valores, useEffect, useEffect para manejar actualizaciones de estado, hacer el llamado a la base de datos, etc..  También el useParams para recuperar el parametro que le pasemos según lo que hayamos puesto, el useContext que en este caso va a servir para que cualquier componente tenga acceso al carrito de donde sea. 
También este proyecto incluye varias dependecias como react-router-dom para el sistema de rutas, react-icons para los iconos, firebase para conectarse con la db, las trabajadas en clase. Pero también incluí una dependecia que es "react-multi-carousel" que en este caso sirve como carrusel para los productos. 

Para correr el proyecto tenes que abrir la consola y situarte en la carpeta del proyecto. Luego hacer un comando llamado npm run seed, esto sirve para cargar todos los productos que estan en el archivo json a una base de datos, donde luego con distintas funcionalidades accederemos a la colección creada y mostraremos los productos por pantalla. Luego de haber hecho ese comando y que los productos se hayan cargado, se escribe el comando npm run dev, para que se levante el proyecto y se inicialice en un host local.
